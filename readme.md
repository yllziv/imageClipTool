遥感影像切割工具（得到影像样本库）
===================================
###用途
将遥感影像按照类别矢量图，切割成很多单一类别的小影像（大小为该类别影像的外接矩形大小，切割后的小影像只有该类别，其他类别为Null），得到影像的样本库

### 目录说明
- myLib 保存python文件
- myLib/entrance.py 实际切割用到的主程序
- other 保存测试过程中的代码，跑程序用不到 
- temp 保存临时文件，主要用来防治内存溢出，例如有10000个图斑，则分为N个，一个python进程只处理10000/N个图斑即可。
- clipTool.cmd 运行程序
- config.py 配置文件
- main.py 程序入口
- python.exe python程序

### 环境
- python 2.6/2.7
- arcgis 10/10.1/10.2 (电脑上一定要装arcgis才可以跑)

### 依赖的库
- arcpy
- numpy
